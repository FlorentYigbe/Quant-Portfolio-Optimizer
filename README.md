# Quant-Portfolio-Optimizer
Starting from adiversified universe of ten assets spanning US equities, European equities, commodities, andfixed income, I construct and evaluate an optimal portfolio using Monte Carlo simulation.The workflow covers :
- Data collectionmvia yfinance
- Individualassetriskanalysis
-  Monte Carlo optimization over 10,000 random portfolios to map the efficient frontier
-  Portfolio comparison across three strategies (MaxSharpe,Min Variance, and Equal Weight)
-  Risk quantification via Value at Risk and Conditional Valueat Risk
